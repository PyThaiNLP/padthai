padthai.GPTNeoFewShot¶

GPT-Neo is a transformer model designed using EleutherAI’s replication of the GPT-3 architecture. https://huggingface.co/EleutherAI

Modules¶

class padthai.GPTNeoFewShot(model_dir: str, model_name: str = 'gpt-neo', device: str = 'cuda', size: str = '125M')¶

Few-Shot Learning using GPT-Neo

Hoempage: EleutherAI/gpt-neo

Thank you code from https://link.medium.com/4FfbALWz8gb

Parameters

  • model_dir (str) – path of model dir

  • model_name (str) – model name (thaigpt-next or gpt-neo)

  • device (str) – device

  • size (str) – model size

Options for size

  • 125M (default) - GPT-Neo 125M / thaigpt-next-125M

  • 1.3B - GPT-Neo 1.3B

  • 2.7B - GPT-Neo 2.7B

Options for model_name

  • thaigpt-next (default) - It is fine-tune the GPT-Neo model for Thai language.

  • gpt-neo
